# Инструменты

Сервер предоставляет типизированные инструменты для частых объектов плюс универсальный `raw_request` для любого другого сервиса/метода — так доступен **весь API v5**.

## Аккаунт и справочники

| Инструмент | Что делает |
| --- | --- |
| `get_account_info` | Данные аккаунта: логин, валюта, тип, страна. |
| `get_quota` | Остаток дневной квоты API (Units: потрачено / осталось / лимит). |
| `get_regions` | Поиск id регионов по названию (нужны для `create_ad_group`). |
| `get_dictionaries` | Справочники (валюты, часовые пояса, константы, …). |

## Кампании, группы, объявления, ключи

| Инструмент | Что делает |
| --- | --- |
| `list_campaigns` / `create_text_campaign` / `update_campaign` / `campaign_action` | Список, создание, обновление (имя, бюджет, **минус-слова**), приостановка/возобновление/архив/удаление. |
| `list_ad_groups` / `create_ad_group` / `update_ad_group` | Список, создание, обновление групп (имя, регионы, **минус-слова**). |
| `list_ads` / `create_text_ad` / `update_text_ad` / `ad_action` | Список, создание, обновление текстовых объявлений; модерация/приостановка/возобновление/архив/удаление. |
| `list_keywords` / `add_keywords` / `set_keyword_bids` / `keyword_action` | Список, добавление, ручные ставки, приостановка/возобновление/удаление фраз. |

## Корректировки ставок

| Инструмент | Что делает |
| --- | --- |
| `get_bid_modifiers` | Чтение корректировок (моб., десктоп, демография, ретаргетинг, гео, видео, SMART TV). |
| `add_bid_modifier` | Добавить корректировку на кампанию или группу. |
| `set_bid_modifiers` | Изменить процент и/или включить/выключить существующие корректировки. |
| `delete_bid_modifiers` | Удалить корректировки по id. |

## Расширения объявлений

| Инструмент | Что делает |
| --- | --- |
| `get_sitelinks` / `create_sitelinks_set` / `delete_sitelinks` | Наборы быстрых ссылок. |
| `get_callouts` / `add_callouts` / `delete_callouts` | Уточнения. |
| `get_vcards` / `create_vcard` / `delete_vcards` | Виртуальные визитки. |
| `get_ad_images` / `get_ad_videos` / `get_creatives` | Чтение библиотек изображений / видео / креативов. |
| `upload_ad_image` | Загрузка изображения (по URL или base64) в библиотеку, возвращает `AdImageHash` для объявления. |

## Статистика и универсальный вызов

| Инструмент | Что делает |
| --- | --- |
| `get_statistics` | TSV-отчёт через сервис Reports. |
| `raw_request` | Прямой вызов любого сервиса/метода (полное покрытие API). Записи требуют `confirmWrite=true`. |

## Деньги и пагинация

Денежные значения в типизированных инструментах — в валюте аккаунта в обе стороны: входные (бюджеты, ставки) автоматически переводятся в микроединицы, а вывод `list_*` конвертируется обратно. `raw_request` работает с сырыми данными (микроединицы).

Списочные инструменты принимают `limit`/`offset` и флаг `autoPaginate`, который проходит все страницы по курсору `LimitedBy`.

## Переменные окружения

| Переменная | Обяз. | По умолчанию | Описание |
| --- | --- | --- | --- |
| `YANDEX_DIRECT_TOKEN` | да | — | OAuth-токен Яндекс Директа. |
| `YANDEX_DIRECT_LOGIN` | нет | — | Заголовок `Client-Login` (агентские аккаунты). |
| `YANDEX_DIRECT_LANG` | нет | `ru` | Язык ответов API (`ru`, `en`, `uk`, `tr`). |
| `YANDEX_DIRECT_SANDBOX` | нет | `false` | Использовать песочницу API. |
| `YANDEX_DIRECT_TIMEOUT_MS` | нет | `60000` | Таймаут запроса, мс. |
| `YANDEX_DIRECT_MAX_RETRIES` | нет | `3` | Повторы при временных ошибках (лимиты, 5xx). |
